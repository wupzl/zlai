# Quickstart: Skill Layer Architecture

## Goal

Use this feature package as the default reference when adding a new skill-layer capability.

## Required Files

- `spec.md`
- `plan.md`
- `tasks.md`

## Strongly Recommended Files

- `research.md`
- `data-model.md`
- `quickstart.md`
- `contracts/skill-contract.md`

## Writing Order

1. Start with `spec.md`
2. Write `plan.md`
3. Write `research.md`
4. Write `data-model.md`
5. Write `contracts/skill-contract.md`
6. Write `tasks.md`

## Review Heuristic

A skill-layer feature is not ready if a reviewer still has to ask:

- What kind of skill is this?
- How is it triggered?
- What exactly does it consume and return?
- What happens if it fails?
- How will operators inspect what happened?
